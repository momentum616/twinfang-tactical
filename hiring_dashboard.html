Role note

Method

This page uses the same salary reference logic as the spreadsheet: country-specific role bands, suggested hiring bands, and a scenario-based effective cost adjustment for fallback comparison. Displayed values follow the site-wide USD / SGD toggle.

  • Raw USD midpoint = midpoint of the reference salary band for that role and country.
  • Effective midpoint = raw midpoint × (1 + relocation premium + cost-of-living premium) ÷ quality-fit score.
  • Recommendation tiers are heuristic and meant for founder discussion, not automatic approval.
  • Region is intentionally left open in the data model for later expansion to China and Taiwan.
